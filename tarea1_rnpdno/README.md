# Tarea 1 – RNPDNO

Análisis del Registro Nacional de Personas Desaparecidas y No Localizadas (RNPDNO).

## Objetivo
Construir una base de datos consolidada por entidad federativa, sexo y grupos de edad
a partir de los datos públicos del RNPDNO.

## Archivos
- `main.rs`: extracción de datos desde la versión pública del RNPDNO usando el crate `reqrnpdno`
- `parteA_rnpdno.R`: limpieza, agregación y transformación de los datos
- `RNPDNO_2024_por_estado_edad_sexo.csv`: base final lista para análisis

## Periodo de análisis
1 de enero de 2024 – 1 de enero de 2025
